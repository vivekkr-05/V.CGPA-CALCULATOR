# V.CGPA-CALCULATOR 🎓

A robust, console-based application designed to help students track their academic performance across multiple semesters using C++.

## ✨ Features
* **Cumulative & Semester Logic:** Calculate SGPA for individual terms and overall CGPA.
* **Persistent Storage:** Saves your records to a local file so you don't lose data.
* **OOP Architecture:** Built using clean Class structures for Students and Courses.
* **Input Validation:** Error handling for invalid grades or credit hours.

## 🛠️ Technical Specs
* **Language:** C++
* **Concepts Used:** Classes/Objects, File I/O (`fstream`), Vectors, and Exception Handling.
